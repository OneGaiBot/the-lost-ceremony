# The Lost Ceremony

Simple band website with typewriter effect and textured font.

## Features

- Black background with centered band name
- Typewriter font with texture (Special Elite)
- Typewriter animation effect
- Responsive design
- Clean, minimalist aesthetic

## Local Development

```bash
npm install
node server.js
```

Visit http://localhost:3004

## Deployment

- **Local:** http://Eliass-Mac-mini.local:3004
